🚀 About the Project

This project is a simple yet powerful language translator built using Python.
It uses the googletrans library to translate text from English → Tamil instantly.

💡 Perfect for beginners learning:

APIs
Python libraries
Automation scripts

📌** Output**
Tamil Translation: வணக்கம், இன்று நீங்கள் எப்படி இருக்கிறீர்கள்?

⚙️ Installation & Setup
# Clone the repository
git clone https://github.com/your-username/language-translator.git

# Navigate into the folder
cd language-translator

# Install dependencies
pip install googletrans==4.0.0-rc1
